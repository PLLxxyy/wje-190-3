# 3D 家居户型设计

一个基于 Web 的 3D 家居户型设计工具，支持拖拽摆放家具、切换房间、调整样式和保存方案。

## 功能特性

- **3D 场景浏览**：自由旋转视角，支持俯视/平视切换
- **房间切换**：客厅、卧室、厨房、卫生间一键切换
- **家具拖拽**：从左侧列表拖拽家具到场景中，松手放置
- **位置调整**：点击选中家具，拖动调整位置
- **旋转控制**：使用属性面板的旋转按钮调整家具角度
- **属性编辑**：右侧面板显示选中家具的尺寸、颜色和材质
- **风格切换**：现代简约、北欧、中式三种整体风格
- **方案管理**：保存多个设计方案，随时切换对比

## 技术栈

- Vite + React 18 + TypeScript
- @react-three/fiber + @react-three/drei + three.js
- zustand 状态管理
- localStorage 数据持久化

## 快速开始

```bash
npm install
npm run dev
```

访问 http://localhost:5173 即可使用。

## 使用说明

1. 从左侧家具列表中拖拽家具到 3D 场景中
2. 点击家具进行选中，右侧属性面板显示详细信息
3. 在属性面板中更换颜色、材质、旋转角度
4. 使用顶部工具栏切换房间和风格
5. 点击「保存方案」保存当前设计，可在右侧面板中切换方案
